MAIN PROGRAM DESIGNER WORK MESSAGE

THE GUEST HOUSE

사다프

루미의 시 〈Guest House〉를 텍스트 대신 숫자의 흐름으로 보여주는 포스터 시리즈입니다. Abjad(압자드)는 아랍·페르시아권의 문자에 숫자 값을 부여하는 전통 체계입니다. 시의 모든 글자를 Abjad 값으로 변환하고, 라틴 숫자와 Abjad 사이 성격을 잇는 ‘새 숫자 형태’를 직접 설계하여 그리드, 범례와 함께 시각화했습니다.
